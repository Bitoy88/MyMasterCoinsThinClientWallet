MyMastercoins Wallet Version 2.0 

1. Create Bitcoin-qt.exe server.
Right click on your desktop
Select New
Select Shortcut
Enter Location of the Item
"C:\Program Files (x86)\Bitcoin\bitcoin-qt.exe" -server
Click Next
Click Finish
Double Click "bitcoin-qt.exe" on your desktop

2. Extract MyMastercoinsWallet2.zip to a folder
ex. c:\temp

3. Run c:\temp\Setup.exe

4. Click [Address] to import all your Address from your Wallet.
